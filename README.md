# Challenge Task | IT-Park

This project was developed as a challenge task within my training course at **IT-Park**. It is a simple yet logically robust counter application.

## 🚀 About the Project

The project allows users to increment numbers, add custom values, and reset the counter. The application is dynamically managed using JavaScript.

### Main Features:

- **Add +1:** Increments the counter by one.
- **Custom Add:** Adds any number entered via the input field to the current value.
- **Reset:** Resets the counter to zero and clears all data.
- **Error Handling:** An `isNaN` check is included to prevent errors when non-numeric values are entered.

## 🛠 Technologies

- **HTML5:** For page structure.
- **CSS3:** For design and interface (custom styles).
- **JavaScript (ES6):** For dynamic logic and event handling.
